# 🤖 פיצ'ר "עבוד עם AI" - מדריך מהיר

## 🎯 מה זה?

כפתור ✨ בכל משימה שמאפשר לעבוד ישירות עם Cursor AI!

---

## 🚀 איך משתמשים?

### שלב 1: רחף מעל משימה
כשמרחפים מעל כרטיס משימה, מופיע כפתור ✨ בפינה השמאלית העליונה.

### שלב 2: לחץ על הכפתור
מה קורה:
1. פרטי המשימה מועתקים ללוח
2. המשימה עוברת ל-"In Progress"
3. מופיעה הודעה: "המשימה הועתקה! 🚀"

### שלב 3: הדבק בצ'אט Cursor
לחץ `Ctrl+V` בצ'אט הזה ותתחיל לעבוד!
