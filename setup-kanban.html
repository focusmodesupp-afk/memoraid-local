🚀 הגדרת Dev Kanban System

מערכת ניהול משימות פיתוח מקצועית

1 ודא שהשרת רץ
הפעל את השרת עם: npm run dev
2 צור את הטבלאות והנתונים
לחץ על הכפתור למטה ליצירת 4 עמודות ו-20 משימות מתכנית העבודה

מה ייווצר?

  • 4 עמודות: Backlog, To Do, In Progress, Done
  • 20 משימות מתכנית העבודה
  • מערכת תגובות מלאה
  • אפשרות להוסיף עמודות נוספות
  • Drag & Drop בין עמודות
  • חיפוש וסינון מתקדם
