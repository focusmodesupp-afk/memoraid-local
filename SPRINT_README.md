## 🎯 מה זה Sprint?

**ספרינט** = מחזור עבודה קצר (בדרך כלל 2 שבועות) עם:
- 🎯 **מטרה ברורה** - מה רוצים להשיג
- 📋 **משימות מוגדרות** - מה צריך לעשות
- 📅 **תאריך יעד** - מתי צריך לסיים
- 📊 **מעקב התקדמות** - איפה אנחנו

---

## 🔄 תהליך עבודה

### 1. Sprint Planning (תכנון)
```
1. צור ספרינט חדש
2. הגדר מטרה: "להשלים אינטגרציה עם Google Calendar"
3. הוסף משימות מהקנבאן
4. הגדר Story Points (הערכת זמן)
5. התחל ספרינט!
```

### 2. Sprint Execution (ביצוע)
```
1. עבוד על המשימות
2. העבר בין עמודות בקנבאן
3. עדכן התקדמות
4. הוסף תגובות
5. עקוב אחר המדדים
```

### 3. Sprint Completion (סיום)
```
1. סיים את כל המשימות
2. לחץ "סיים ספרינט"
3. משימות שלא הושלמו חוזרות ל-Backlog
4. סקור מה למדת
5. תכנן ספרינט הבא!
```

## 💡 טיפים

### Sprint Planning
- 🎯 הגדר מטרה ברורה ומדידה
- 📋 אל תעמיס יותר מדי משימות
- ⏱️ הערך זמן באופן ריאליסטי
- 👥 שתף את הצוות בתכנון

### Sprint Execution
- 📅 עדכן סטטוס יומי
- 💬 תקשר עם הצוות
- 📊 עקוב אחר ה-Burndown
- 🚀 התמקד במטרה

### Sprint Completion
- ✅ סיים רק כשהכל מוכן
- 📝 תעד מה למדת
- 📈 חשב Velocity
- 🔄 שפר את הספרינט הבא
