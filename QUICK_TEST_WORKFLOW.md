# 🧪 בדיקה מהירה של הזרימה

## ⚠️ לפני שמתחילים

### 1. וודא שהשרת רץ
```powershell
cd c:\Users\USER\OneDrive\Documentos\memoraid-local
npm run dev:server
```

### 2. צור את כל הטבלאות (בדפדפן)
```
http://localhost:3006/api/health/create-dev-kanban-tables
http://localhost:3006/api/health/create-sprint-tables
http://localhost:3006/api/health/update-admin-password
```

### 3. צור ספרינט פעיל
גש ל: `http://localhost:5173/admin/settings/work-plan`

לחץ על: **"ספרינט חדש"**

מלא:
- שם: `Sprint 12: Calendar Integration`
- מטרה: `להשלים אינטגרציה עם Google Calendar ו-Outlook`
- תאריך התחלה: `2026-02-20`
- תאריך סיום: `2026-03-06`

לחץ: **"צור ספרינט"**

### 4. הפעל את הספרינט
בדף הספרינט, לחץ על: **"התחל ספרינט"**

## ✅ בדיקת הזרימה

### שלב 1: Roadmap → Sprint
1. גש ל: `http://localhost:5173/admin/settings/work-plan`
2. גלול למטה ל-"Roadmap - מפת דרכים"
3. מצא את **"Phase 5 - Integrations"** (צריך להיות בכחול)
4. רואה את הטקסט: **"לחץ לצפייה בספרינט"**
5. **לחץ על כל האיזור הכחול**
6. ✅ אתה אמור לעבור לדף הספרינט!

### שלב 2: Sprint → Kanban
1. אתה בדף הספרינט (`/admin/sprints/...`)
2. רואה את הכפתור הסגול: **"פתח בקנבאן ✨"**
3. **לחץ על הכפתור**
4. ✅ אתה אמור לעבור לקנבאן עם באנר סגול!

### שלב 3: Kanban → AI
1. אתה בקנבאן (`/admin/dev/kanban?sprint=...`)
2. רואה באנר סגול: **"מציג משימות מספרינט"**
3. בחר משימה כלשהי
4. **העבר עכבר על המשימה**
5. רואה כפתור ✨ בפינה השמאלית העליונה
6. **לחץ על כפתור ✨**
7. ✅ אתה אמור לראות הודעה: "✨ המשימה הועתקה!"
8. המשימה עברה ל-"In Progress"
9. פתח Cursor Chat והדבק (Ctrl+V)
10. ✅ רואה את כל פרטי המשימה בפורמט מקצועי!

## 🐛 פתרון בעיות

### בעיה: Phase 5 לא קליקאבילי
**סיבה:** אין ספרינט פעיל

**פתרון:**
1. צור ספרינט חדש (שלב 3 למעלה)
2. הפעל אותו (לחץ "התחל ספרינט")
3. רענן את דף Work Plan
4. עכשיו Phase 5 צריך להיות קליקאבילי

### בעיה: לחצתי על Phase 5 ולא קרה כלום
**פתרון:**
1. פתח את Developer Console (F12)
2. לחץ שוב על Phase 5
3. בדוק אם יש שגיאות
4. אם יש שגיאת 404 - הספרינט לא קיים
5. צור ספרינט חדש והפעל אותו

### בעיה: הקנבאן ריק
**פתרון:**
```
http://localhost:3006/api/health/create-dev-kanban-tables
```
זה יצור 20 משימות ברירת מחדל

### בעיה: אין כפתור "פתח בקנבאן"
**פתרון:**
1. וודא שאתה בדף ספרינט (`/admin/sprints/...`)
2. רענן את הדף
3. הכפתור צריך להיות בראש הדף, בצד ימין

## ✨ טיפים

### טיפ 1: צבעים
- **כחול** = Phase בביצוע (קליקאבילי)
- **ירוק** = Phase שהושלם
- **אפור** = Phase מתוכנן
- **סגול** = הקשר של ספרינט

### טיפ 2: אייקונים
- **⏳** = בביצוע
- **✓** = הושלם
- **○** = מתוכנן
- **✨** = AI / קנבאן

### טיפ 3: ניווט
- יש באנר סגול? → אתה בהקשר של ספרינט
- רוצה לצאת? → לחץ X או "חזרה לספרינט"
- רוצה לראות הכל? → הסר את `?sprint=...` מה-URL
