# 🤖 פיצ'ר "עבוד עם AI" - Phase 1

## 🎯 מה זה?

כפתור מהפכני בכל כרטיס משימה שמאפשר לעבוד ישירות עם Cursor AI על המשימה!

## ✨ איך זה עובד?

### 1. הצג את הכפתור
כשמרחפים מעל כרטיס משימה, מופיע כפתור **✨ Sparkles** בפינה השמאלית העליונה.

### 2. לחץ על הכפתור
מה קורה:
1. ✅ **פרטי המשימה מועתקים ללוח** בפורמט מובנה ומסודר
2. ✅ **המשימה עוברת אוטומטית ל-"In Progress"**
3. ✅ **הודעה מופיעה**: "המשימה הועתקה! הדבק בצ'אט Cursor והמשימה עברה ל-In Progress 🚀"

### 3. הדבק בצ'אט Cursor
פשוט תלחץ `Ctrl+V` בצ'אט הזה ותתחיל לעבוד!
